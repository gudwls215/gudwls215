## 🚀 실시간 미리보기 방법

### VS Code에서:
1. `Ctrl + Shift + P` (명령 팔레트)
2. "Markdown: Open Preview to the Side" 검색
3. README.md 파일과 미리보기를 나란히 보기

### 브라우저에서:
1. GitHub 레포지토리 방문
2. README.md 파일 클릭
3. "Preview" 탭 클릭
